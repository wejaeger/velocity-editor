Provides basic support to Velocity's *.vm and *.vsl files. Syntax coloring and basic error highlighting is achieved with a lexer and parser based on Apache Velocity's 1.6.2 specification and compiled with JavaCC 5.0.

Current Version:

Supports syntax- and error highlighting.

Konwn issuses:

  • Syntax color is lost when editing variables, properties and methods. A workaround is to delete and insert the preceeding $ sign.
  • Error recovery is rudimentary. At the present time the reovery point is allways the next LPAREN. Needs to be revised.
  • A macro call is neither syntax colored nor is the argument count checked.
  • No syntax coloring for directives #include, #parse, #evaluate, #define
  • The litte square icon on the upper right corner of the editor margin turns neiter to red nor to green (no idea why)
